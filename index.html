The Product

Hand crafted from the finest bits

Tribe.im is a real-time communications service for teams. It is currently under development. The roadmap includes many awesome features.

  • Messaging

    • One-on-one instant messaging
    • Conference chat rooms
    • Federated to other XMPP services and transports

  • Profiles

    • Business card
    • Public/private information
    • Integrates with LinkedIn, Twitter, Gravatar, etc.

  • Pubsub Streams

    • Subscribe to data feeds to receive real-time instant notifications
    • Connects to RSS, email, CRM, ERP, etc.

  • File Sharing

    • Collaborate using team-based file repositories
    • Integrates with Dropbox (AWS S3)

  • Voice and Video

    • Live VOIP calls and video streams
    • Peer to peer for low latency, high quality
    • Bridge to regular telephone networks

The Technology

Sufficiently advanced magic

Tribe.im uses open standards such as XMPP for its backend infrastructure. Since there are many high quality XMPP servers available as free software, this allows Tribe.im to focus on a high quality user experience.

The frontend of Tribe.im is a pure Javascript application built using the latest and greatest technologies available via HTML5 and CSS3. Tribe.im takes the long term view of standards compliance over cross platform compatibility.

Creation of Tribe.im takes place on GitHub and Dropbox. To join the effort, take up an Issue and send a Pull Request when you’re done.
